Add graphite graphs to your site

== Description ==

Easily add graphite graphs to your admin area and to your sidebar. Configure your color scheme, metrics and display settings with a few clicks to share anything that can be measured on graphite with your users or administrators. Note: This plugin requires an external graphite host (http://graphite.wikidot.com or https://www.hostedgraphite.com).

== Installation ==

Automatic Installation:

1. Go to Admin - Plugins - Add New and search for "Graphite Graphs"
2. Click on Install
3. Click on Activate

Manual Installation:

1. Download graphite-graphs.zip
2. Unzip and upload the "graphite-graphs" folder to your "/wp-content/plugins/" directory
3. Activate the plugin through the "Plugins" menu in WordPress

Configuration:

Configure your dashboard graph settings directly on your dashboard and configure your widgets in Appearance -> Widgets
